Mulesoft Accelerator Repository for reusability and implement Native Checkout API for Gemini AI

# Hi, I'm Viplove Goswami 👋
## 🚀 Featured Project: Agentic Commerce Experience

This project demonstrates a full **API-Led Connectivity** architecture using MuleSoft and Gemini.

### 🏗️ Architecture Overview


| Layer | Repository | Responsibility |
| :--- | :--- | :--- |
| **Experience (EAPI)** | https://github.com/viplgoswami1/mulesoft-gemini-agentic-eapi (#) | Gemini-native checkout endpoints (UCP compliant). |
| **Process (PAPI)** | [[Link to PAPI Repo](https://github.com/viplgoswami1/mulesoft-gemini-agentic-eapi)](#) | Orchestrating order fulfillment and inventory logic. |
| **System (SAPI)** | [Link to SAPI Repo](#) | Connectivity to SAP, Salesforce, and Payment Gateways. |

### 🛠️ Tech Stack
- **MuleSoft:** Anypoint Studio 7.x, RAML 1.0, DataWeave 2.0
- **AI:** Google Gemini (Agentic workflows)
- **Deployment:** CloudHub 2.0 / GitHub Actions
